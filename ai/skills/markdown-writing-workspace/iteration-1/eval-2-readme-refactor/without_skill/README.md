# DataTool

A command-line tool for processing and transforming data files in multiple formats.

## Features

- **transform**: Convert data between CSV, JSON, and XML formats
- **validate**: Check data quality and detect common issues
- **export**: Export data in various formats for integration with other tools
- **merge**: Combine multiple files into a single output

## Requirements

- Python 3.9 or higher
- macOS or Linux

## Installation

```
pip install datatool
```

## Usage

Run the tool with a command and optional flags:

```
datatool <command> [options]
```

Each command supports its own set of options. See `datatool <command> --help` for details.
